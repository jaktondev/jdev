About CrabCLI

CrabCLI, or simply crab, is a CLI framework/shard for making prettier and better command line applications for the Crystal Programming language. It is inspired by the rich library from Python.

WHY? Because I love CLI apps and I really liked Crystal, but there was no shard for prettier CLI's when I began to learn it. I would love CLI's to be made with such an easy and powerful language.

Dependencies? Crab does not have any other Crystal dependencies, but it does make use of the stty command to get the terminal size. Most Linux distros have it installed by default, so make sure you have it too.

Works on Windows? As Crystal Lang is not fully supported on Windows, I have not tested it, so I have no idea and do not recommend it. If Crystal is fully ported to Windows, I will try to make it work.

How to install? Add to your shard.yml file: 

        
dependencies:
  crab:
    github: jaktondev/crab

Then run: shards install
